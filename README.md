# inteligencia-artificial-tarea
Implementación del algoritmo A*, heurística y teoría de juegos en Python.

## Algoritmo A*

El algoritmo A* es un algoritmo de búsqueda utilizado en inteligencia artificial
para encontrar el camino más corto entre dos nodos dentro de un grafo.

Este algoritmo combina el costo real del camino recorrido con una estimación
del costo restante hasta el objetivo, lo que permite encontrar rutas óptimas
de forma eficiente.

Se usa mucho en:
- videojuegos
- robótica
- navegación
- sistemas de rutas

## Heurística

Una heurística es una función que permite estimar la distancia o costo
desde un nodo actual hasta el nodo objetivo.

Sirve para guiar al algoritmo de búsqueda hacia la solución más rápida.

Ejemplo:
Si queremos ir de una ciudad a otra, una heurística podría ser la
distancia en línea recta entre ambas ciudades.


## Teoría de Juegos

La teoría de juegos es una rama de las matemáticas y la economía
que estudia las decisiones estratégicas entre varios participantes.

Se utiliza para analizar situaciones donde el resultado depende
de las decisiones de más de un agente.

Aplicaciones:
- economía
- inteligencia artificial
- negociación
- estrategia empresarial
